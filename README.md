# 🐋 Fierce Orcas — Coach's Playbook

A complete volleyball coaching toolkit built for a 7th–9th grade travel team. 13 interactive tools — open any file in your browser and coach.

**🚀 Open `dashboard.html` to access everything from one screen.**

---

## 📂 Tools

| File | Tool | Category |
|------|------|----------|
| `dashboard.html` | **Coach Dashboard** — home screen, practice timer, tip of the day | Home |
| `volleyball-season-plan.html` | **Season Practice Plan** — 12-week, 4 phases | Planning |
| `drill-board.html` | **Drill Board** — 24 GMS drills, filters, timer *(video links updated)* | Planning |
| `volleyball-eval-system.html` | **Player Evaluation System** — 4 pillars, radar charts | Evaluation |
| `tryout-drill-system.html` | **Tryout Drill System** — 7 stations, live scoring | Evaluation |
| `rotation-iq.html` | **RotationIQ** — starting lineup & rotation builder | Evaluation |
| `volleyball-fix-serve-receive.html` | **Fix: Serve Receive & Ball Fear** | Skill Training |
| `jump-training.html` | **Jump Higher Program** — 12-week plyometric | Skill Training |
| `volleyball-blocking-guide.html` | **Teaching Blocking** — technique + 8 drills | Skill Training |
| `volleyball-transition-guide.html` | **Coaching Transition** — 5 phases + plan | Skill Training |
| `62-rotation-academy.html` | **6-2 Rotation Academy** — interactive player game | Game Systems |
| `serve-zone-game.html` | **SERVE ZONE** — serving mental reps game | Games |
| `spike-serve-game.html` | **SPIKE** — POV first-person serve trainer | Games |
| `FierceOrcas_Roster.xlsx` | **Team Roster** — all 13 players + emergency contacts | Admin |
